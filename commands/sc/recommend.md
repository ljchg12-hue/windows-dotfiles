# SuperClaude Intelligent Command Recommender

## Command
```bash
/sc:recommend [user request] --options [flags]
```

## Multi-language Support
- Turkish, English, and cross-language detection
- Universal pattern recognition

## SuperClaude Command Map
- **ml_category**: /sc:analyze, /sc:design with --seq --c7
- **web_category**: /sc:build, /sc:design with --magic --c7
- **api_category**: /sc:design --api --ddd, /sc:build --tdd
- **debug_category**: /sc:troubleshoot, /sc:analyze with --seq
- **performance_category**: /sc:analyze --performance, /sc:improve
- **security_category**: /sc:scan --security --owasp
- **create_category**: /sc:build --feature, /sc:design
- **test_category**: /sc:test --coverage --e2e
- **improve_category**: /sc:improve --quality, /sc:cleanup
- **learning_category**: /sc:document, /sc:analyze with --persona-mentor

## Features
1. **Multi-language Support**: Turkish, English, auto-detection
2. **Project Context Detection**: File system analysis
3. **Real-Time Streaming Mode**: Continuous project tracking
4. **Alternative Recommendations**: Multiple options with comparison
5. **Time/Budget Estimation**: Intelligent estimation
6. **Smart Flag Recommendations**: Context-based flags
7. **Community Patterns**: Data from successful projects

## Usage
```bash
/sc:recommend "I want to do something"
/sc:recommend "new React project" --estimate --alternatives
/sc:recommend --stream "developing e-commerce site"
```
