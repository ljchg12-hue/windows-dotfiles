# Deep Research

## Purpose
Conduct comprehensive research using multiple sources, cross-verification, and systematic analysis to provide accurate, well-sourced information.

## Activation Keywords
- deep research, comprehensive research
- fact check, verify, validate
- multi-source, cross-reference
- investigate, analyze thoroughly
- research report, findings

## Core Capabilities

### 1. Multi-Source Research
- Web search aggregation
- Academic sources
- Official documentation
- Expert opinions
- Primary sources

### 2. Verification
- Cross-reference facts
- Source credibility assessment
- Bias detection
- Recency validation
- Citation tracking

### 3. Analysis
- Pattern identification
- Trend analysis
- Gap detection
- Contradiction resolution
- Synthesis

### 4. Documentation
- Structured findings
- Source attribution
- Confidence levels
- Limitations noted
- Actionable insights

### 5. Quality Assurance
- Fact accuracy check
- Source diversity
- Completeness assessment
- Objectivity review

## Research Methodology

```
1. Define Scope
   → Clear research questions
   → Success criteria
   → Time/depth constraints

2. Gather Sources
   → Primary sources (official docs, APIs)
   → Secondary sources (articles, reports)
   → Tertiary sources (summaries, reviews)
   → Minimum 5+ independent sources

3. Verify & Cross-Reference
   → Fact triangulation
   → Date verification
   → Author credibility
   → Source independence

4. Analyze & Synthesize
   → Pattern extraction
   → Contradiction resolution
   → Gap identification
   → Insight generation

5. Document Findings
   → Executive summary
   → Detailed findings
   → Source citations
   → Confidence ratings
   → Limitations
```
