# 작업 허락 체크리스트
**ABSOLUTE RULES #5 참조**

## 빠른 체크 (Write/Edit/Bash 전)
```
1. 작업요약: [1-2줄]
2. 범위: 파일 X개, 도구 Y개, AI Z개, MCP N개, 위험도
3. 모드: 오토/부분수락/수동
4. 허락대기
```

## 예외 (즉시 실행)
Read, Grep, Glob, Bash:ls/cat/pwd/echo/which

## 위험도 기준
- **낮음**: 읽기전용, 영향최소
- **중간**: 파일수정, 복구가능
- **높음**: 삭제/이동, 시스템변경

## 모드 선택
| 모드 | 언제 | 특징 |
|------|------|------|
| 오토 | 낮음~중간 | 전체 자동 |
| 부분수락 | 중간~높음 | 단계별 확인 |
| 수동 | 높음 | 각 작업 확인 |

## 범위 초과 시
→ 재허락 필수 (추가 작업 목록 제시)

## 레포팅 (작업 완료 시)
- 사용도구 (이름+횟수)
- 사용AI (Cloud/Ollama Cloud/Local)
- 사용MCP, Agent
- 통계 (묶음수, 시간, 파일수)
