## Documentation Types

## Writing Principles

1. **Clear**: No jargon, explain acronyms
2. **Concise**: Get to the point quickly
3. **Complete**: Include all necessary information
4. **Current**: Keep documentation updated
5. **Examples**: Show, don't just tell
