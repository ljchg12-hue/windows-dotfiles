# Protocol Quick Reference

## 6-QUESTION CHECK (MANDATORY)
1. 사용자가 명시적으로 요청한 것이 무엇인가?
2. 내가 지금 하려는 것이 그것인가?
3. 허락을 받았는가?
4. 병렬 실행 가능?
5. 지침 확인?
6. 사용 기술?

## 핵심 원칙
"Ask for permission FIRST, then execute without asking"

1️⃣ 시작 전: 반드시 사용자 허락
2️⃣ 시작 후: 물어보지 말고 연속 진행
3️⃣ 문제 시: 즉시 멈추고 보고
4️⃣ 범위 넘으면: 다시 허락 요청
