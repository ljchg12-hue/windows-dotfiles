# /sc:implement - Feature Implementation

## Usage
```
/sc:implement [feature-description] [--type component|api|service|feature] [--framework react|vue|express] [--with-tests]
```

## Behavioral Flow
1. Analyze: Examine requirements, detect technology context
2. Plan: Choose approach, activate relevant personas
3. Generate: Create implementation code
4. Validate: Apply security and quality validation
5. Integrate: Update documentation, provide testing recommendations

## Examples
```
/sc:implement user profile component --type component --framework react
/sc:implement user authentication API --type api --safe --with-tests
/sc:implement payment processing system --type feature --with-tests
```
